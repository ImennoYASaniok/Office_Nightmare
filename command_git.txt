1) git push --force origin 'название ветки'
Переносит ветку origin 'название ветки' (удалённый репозиторий) к ветке 'название ветки' (локальный репозиторий)

2) git tag -d 'имя тега'
Удаление тега из локального репозитория

3) git push origin --delete имя тега'
Удаление тега из удалённого репозитория
